**EduSphere Smart Portal** is an interactive web application built with Streamlit that allows educational institutions to manage student records, courses, and grades in a simple and intuitive way. This portal is designed for Admins, Teachers, and Students to streamline academic operations.

🌟 Features
- User Authentication:
Login system with predefined users: Admin, Teacher, and Student.
- Student Management:
  - Add new students with unique IDs
  - Store student profiles and GPA
- Course Registration:
 1. Register students for STEM courses (Python, Cyber Security, Mathematics)
 2.  Maintain a list of courses per student
- Grade Calculator:
 1.  Input scores for students and calculate grades (A+, A, B, C, Fail)
 2. Automatically update student GPA
- View Records:
  View detailed student information including registered courses and GPA
- Mini Database:
  Uses st.session_state to maintain data during the session
